# Earl Noble Savings (ENS)

Premium Ghanaian susu marketing site and member portal: register, log in, and track savings in the browser (demo storage via `localStorage` until a backend is connected).

Built with [Next.js](https://nextjs.org) (App Router), TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Copy `.env.example` to `.env.local` and set your public site URL and contact fields before production.

```bash
npm run build
npm run start
```
